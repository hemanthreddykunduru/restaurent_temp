# Sangem Hotels Restaurant Website

## Quick Start

Visit the application at: **http://localhost:3000**

## Location

Project directory: `h:\website_temp\sangem-hotels`

## Features

Complete restaurant website with:
- Multi-branch support (5 locations)
- Menu system (20 dishes)
- Shopping cart
- Multiple payment methods
- Table booking
- Feedback system
- Excel data file generated

## Data File

Menu and branch data: `sangem-hotels-data.xlsx`
